## 📁 How to download the Project 
<br/>

```bash 
    #clone the repository
     $ git clone https://github.com/VidalJunior05/drag-and-drop-javasctip.git
    
    # Enter directory
    $ cd arrastar-e-soltar-javascript

    # install dependencies
    $ yarn install

    # Initialize project
    $ yarn start
```
